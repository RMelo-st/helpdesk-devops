# HelpDesk DevOps

Projeto acadêmico para demonstração prática de conceitos DevOps utilizando:

- Python Flask
- PostgreSQL
- Docker
- Docker Compose
- GitHub

## Objetivo

Demonstrar containerização, persistência de dados, versionamento e publicação de releases.

## Estrutura

- API REST para gerenciamento de chamados
- Banco PostgreSQL
- Containers Docker
